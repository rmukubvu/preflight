AI Diagnosis Provider
Used to explain assertion failures and suggest fixes. auto picks the best available provider based on your credentials.
Anthropic Claude
Highest quality diagnosis. Uses claude-haiku by default.
Get your key at console.anthropic.com
Use claude-sonnet-4-6 for deeper multi-service analysis.
OpenAI
Also supports Azure OpenAI via the base URL override.
Get your key at platform.openai.com
Use gpt-4o for deeper analysis.
AWS Bedrock
Uses your existing AWS credentials — no new API key required. Works with any credential chain: ~/.aws/credentials, IAM roles, env vars.
amazon.nova-lite-v1:0 is generally available. Claude on Bedrock requires separate model access approval.
Ollama
Free, runs locally, no data leaves your machine. Works air-gapped.
Any pulled model works: llama3.2, mistral, gemma4, etc.
Floci — Local AWS Emulator
Floci is a Quarkus-native AWS emulator. Starts in 24ms, uses 13MB RAM, supports 100% of AWS SDK operations.
